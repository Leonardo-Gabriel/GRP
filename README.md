## Descricao resumida do sistema

O GRP - Gestao Rural Pro e uma aplicacao de gerenciamento rural criada para auxiliar na organizacao de informacoes agricolas. Nesta Etapa 2, foi implementado um fluxo funcional completo de gestao de culturas agricolas, permitindo cadastrar, consultar, editar e excluir culturas vinculadas a talhoes de uma propriedade rural.

## Fluxo funcional implementado

1. Visualizacao do dashboard inicial.
2. Cadastro de cultura agricola.
3. Consulta/listagem das culturas cadastradas.
4. Edicao dos dados de uma cultura.
5. Exclusao de uma cultura com confirmacao.
6. Atualizacao automatica dos indicadores do dashboard.

## Tecnologias utilizadas

- React
- JavaScript
- Vite
- HTML
- CSS
- LocalStorage para simulacao de persistencia de dados

## Instrucoes para execucao

Acesse a pasta do codigo-fonte:

```bash
cd codigo-fonte
```

Instale as dependencias:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

Acesse no navegador o endereco exibido no terminal. Geralmente:

```text
http://localhost:5173
```

## Funcionalidades implementadas

- Tela inicial com resumo do sistema.
- Dashboard com indicadores de culturas cadastradas, area total e culturas em desenvolvimento.
- Formulario de cadastro de culturas.
- Validacao simples de campos obrigatorios e area maior que zero.
- Listagem de culturas em tabela.
- Edicao de culturas cadastradas.
- Exclusao de culturas com confirmacao.
- Persistencia local dos dados no navegador.
- Layout responsivo.
